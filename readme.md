# Events Api

## Routes (with Appropriate Controllers)

### /users

- Get all users
- Post user
- Update user
- Delete user
- Get one by user id
- Get users by group id

### /groups

- Get all groups
- Get group by group id
- Get group by user(admin) id
- Post group
- Update group
- Delete group

### /events

- Get all events
- Get events by events id
- Get events by group id
- Get events by user id
- Post event
- Update event
- Delete event
